Basic Structure of a HTML code

Every language has its basic structure. So do HTML.

Here's an example of the basic structure of a HTML code.


  <!DOCTYPE html>
  <html lang="en">
      <head>...</head>
      <body>...</body>
  </html>
What to write inside a head?

Key elements in head:

  • Title and Encoding
  • Viewport for responsiveness
  • CSS link
  • JavaScript

  <head>
    <title>My Webpage</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="A great webpage">
    <meta property="og:title" content="My Webpage">
    <link rel="stylesheet" href="styles.css">
    <script src="script.js" defer></script>
  </head>
What to write inside a body?

Key elements in body:

  • Headings and Text content
  • Images and Media
  • Links and navigations
  • Lists (ordered and unordered)
  • Forms and User Input
  • Buttons and Interactivity
  • Footer

  <body>
    <h1>Welcome to My Website</h1>
    <p>This is a sample paragraph.</p>

    <img src="image.jpg" alt="A beautiful view">
    <video controls>
        <source src="video.mp4" type="video/mp4">
    </video>

    <nav>
        <a href="home.html">Home</a>
        <a href="about.html">About</a>
    </nav>

    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>

    <form action="submit.php" method="post">
        <input type="text" placeholder="Enter your name">
        <button type="submit">Submit</button>
    </form>

    <button onclick="alert('Hello!')">Click Me</button>

    <footer>
        <p>© 2025 My Website</p>
    </footer>
</body>
Basic CSS

A few concept of CSS:

  • Selectors (Targeting elements)
  • Colors and backgrounds
  • Text Styling (Font, Size,and Alignment)
  • Box Model (Margin, Padding, Border)
  • Flexbox for Layout
  • Hover effects and Transitions

  /* 1. Selectors */
  body {
    font-family: Arial, sans-serif;
  }

  /* 2. Colors & Backgrounds */
  h1 {
    color: blue;
    background-color: lightgray;
  }

  /* 3. Text Styling */
  p {
    font-size: 16px;
    text-align: center;
  }

  /* 4. Box Model */
  div {
    width: 200px;
    padding: 10px;
    margin: 20px;
    border: 2px solid black;
  }

  /* 5. Flexbox for Layout */
  .container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
  }

  /* 6. Hover Effects & Transitions */
  button {
    background-color: green;
    color: white;
    padding: 10px 20px;
    border: none;
    transition: background-color 0.3s;
  }

  button:hover {
    background-color: darkgreen;
  }
Conclusion

HTML provides the structure of a webpage, while CSS enhances its style and layout. Together, they create visually appealing and functional websites. Mastering both allows you to build responsive, user-friendly designs.
